AURA Pro

Role Frontend Developer
Timeline Product sprint
Year 2026

Overview

AURA Pro is a single-page product landing site for wireless headphones featuring interactive 3D product viewing and cinematic motion.

The page is built to feel premium while staying lightweight, with a fallback viewer for devices that cannot render WebGL.

The Challenge

The challenge was creating a high-impact product story without heavy assets or performance drops on mid-range devices.

The experience also needed to degrade gracefully for users who cannot load 3D content.

The Solution

I built a scroll-led layout with reveal animations, a 3D viewer for the hero section, and an SVG fallback when WebGL is unavailable.

Subtle motion cues guide attention across feature cards, spec counters, and the final call-to-action.

Tech Stack

HTML CSS GSAP Three.js Lucide

Key Features

• Hero 3D Viewer — Interactive GLB model with SVG fallback.

• Scroll Progress — Visual indicator and reveal animations.

• Feature Cards — Clear product benefits with motion cues.

• Spec Counters — Animated stats for fast scanning.

• Responsive Layout — Mobile navigation and adaptive spacing.
