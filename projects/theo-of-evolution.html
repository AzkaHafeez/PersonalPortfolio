Theo of Evolution

Role Frontend Developer
Timeline Storytelling build
Year 2026

Overview

Theo of Evolution is a scroll-driven storytelling site about the evolution of life, built with immersive scenes and theme selection.

The experience blends narrative pacing with motion design to keep long-form content engaging and easy to follow.

The Challenge

Long-form storytelling can feel heavy if motion and typography do not support the narrative structure.

The site needed smooth scroll behavior, reusable motion helpers, and theming that did not slow down the experience.

The Solution

I built a modular project structure with reusable motion utilities and pinned sections to control pacing as the story unfolds.

React and Vite keep the build fast, while GSAP and ScrollTrigger handle the scene transitions and UI feedback.

Tech Stack

React 19 Vite 8 GSAP ScrollTrigger Tailwind CSS Custom CSS

Key Features

• Scroll-triggered Sections — Pinned scenes and narrative pacing.

• Theme Selection — Custom theming for immersive mood shifts.

• Smooth Transitions — GSAP-driven reveals and scene changes.

• Modular Architecture — Reusable components and motion helpers.

• Responsive Layout — Designed for desktop and mobile.
