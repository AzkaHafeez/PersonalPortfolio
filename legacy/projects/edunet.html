Edunet

Role Flutter Developer
Timeline Prototype build
Year 2025

Overview

Edunet is a local-first, offline classroom platform built with Flutter. It lets teachers and students communicate, share files, and run calls without internet.

The app runs on Android devices that support BLE and Wi-Fi Direct.

The Challenge

Most classroom tools assume stable internet, which breaks down in low-connectivity environments and during on-campus demos.

The challenge was building reliable peer-to-peer communication with clear roles for teachers (hosts) and students (clients).

The Solution

We used Google Nearby Connections for offline discovery and data transfer, paired with WebRTC for audio and video calls.

A star topology with application-layer routing lets the teacher node rebroadcast student messages, simulating a mesh-like classroom network.

Tech Stack

Flutter Provider Hive nearby_connections flutter_webrtc

Key Features

• Offline Connectivity - Google Nearby Connections for local discovery.

• Real-time Chat and File Sharing - Lightweight classroom messaging.

• Audio and Video Calls - WebRTC-based communication without internet.

• Role-based Access - Teachers host, students join as clients.

• Mesh-like Rebroadcasting - Teacher node forwards student messages.

• Android Support - Requires BLE and Wi-Fi Direct capable devices.
